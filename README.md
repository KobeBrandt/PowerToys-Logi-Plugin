
# PowerToys-Logi-Plugin

An unofficial Logi Plugin that executes the PowerToy shortcut for you.


## Requirements

To use this plugin you need:

- PowerToys installed
- A compatible device:
    - Logitech Creative Console
    - Logitech Actions Ring (Can be used on all MX devices)
    - Loupedeck device with recent driver (Not tested)
